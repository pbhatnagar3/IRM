Instructions for setting up the SPF search tool:

ITEMACTION
SPF search tool
  1. Install AMP: Apache | MySQL | PHP
  2. AMP on Ubuntu:
    sudo apt-get install apache2
    sudo apt-get install php5 libapache2-mod-php5
    sudo apt-get install mysql-server
  3. Unzip SPFsearch.zip: place in /var/www/html directory
  4. Login to MySQL and Create Database: - assume user=root, pass=root, database=its
    mysql -u root -proot
    mysql>> create database its; [ENTER]
    quit; [ENTER]
  5. Load SPF Database:
    mysql -u root -D its -proot < sql/SPF.sql
  6. Test:
    • http://localhost/search/search.php
    • http://localhost/search/tables.php
